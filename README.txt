**********************Steps to run the project files**********************

step 1. Extract the files in a new folder.
step 2. Open the folder in an IDE.
step 3. Open cmd or shell and navigate to this folder and enter command ' npm install ' (without quotes)
step 4. Once the installation is finished, enter ' npm start ' in cmd, in the same folder.
step 5. Open http://localhost:3000 in your browser.
